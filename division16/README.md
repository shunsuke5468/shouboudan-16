# 第16分団 一班 スケジュール管理アプリ

令和8年度(2026年度) 小田原消防団 第16分団 一班のスケジュール・連絡管理アプリです。

## 機能

- 📋 ポンプ・自主点検の年間日程表
- 📅 行事・講習・健康診断の予定
- 👥 班員一覧
- 📢 班会議事録・連絡事項

## 開発

```bash
npm install
npm run dev
```

## デプロイ

`main` ブランチにプッシュすると GitHub Actions が自動でビルド・デプロイします。
